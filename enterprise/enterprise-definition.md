# Enterprise Definition

## Purpose

Chores Cooperative is a household-oriented system supporting task management,
cooperation, accountability, and family coordination.

## Enterprise Scope

The enterprise consists of:
- the software system itself
- the AWS infrastructure hosting the system
- the development and operational environment
- the household members using the application
- the development capabilities required to evolve and maintain the system

## Core Enterprise Units

- Chores Cooperative application
- AWS account and cloud infrastructure
- Household users
- Software development and operations capability

## Soft Enterprise Units

- Development tooling
- Monitoring and backup systems
- Local development devices

## Extended Enterprise

- AWS services
- Internet and DNS infrastructure
- Third-party ecosystems and providers

## Stakeholders

- Application users
- System creator and operator
- Future contributors (potentially)

## Architectural Scope

The architecture repository covers:
- application architecture
- cloud infrastructure
- operational processes
- migration strategies
- architectural principles
- event architecture
